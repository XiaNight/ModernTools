# CommonProtocol

Shared device-protocol tooling: **Bus Hound** (USB packet inspection), device log, relay control, a quick-action command editor, and a USBPcap handler. Protocol definitions live in `Protocols/` (`GetInfo.cs`, `KBLayoutNation.cs`).

This is general-purpose protocol tooling used across peripheral types — but it is still a feature, not `Base`. Keep it here.
